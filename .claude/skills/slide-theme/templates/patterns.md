## `cover` — カバースライド

**用途**: プレゼンテーションの表紙
**背景**: グラデーション
**テキスト色**: 白

### Marpコード例

```markdown
<!-- _class: cover -->

# プレゼンタイトル

## サブタイトル・登壇者名
```

### 容量制限

| 要素 | 上限 |
|:-----|:-----|
| h1 | {{COVER_H1_LIMIT}}文字（1行） |
| h2 | {{COVER_H2_LIMIT}}文字（1行） |

---

## `bridge` — ブリッジスライド

**用途**: セクション転換、感情的インパクト
**背景**: グラデーション
**テキスト色**: 白

### Marpコード例

```markdown
<!-- _class: bridge -->

# 1行メッセージ
```

### 容量制限

| 要素 | 上限 |
|:-----|:-----|
| h1 | {{BRIDGE_H1_LIMIT}}文字（1行、`<br>`で改行可） |

---

## `content` — コンテンツスライド

**用途**: 箇条書きでの説明
**背景**: 白
**ヘッダーバー**: あり（`<!-- _header: "セクション名" -->`）

### Marpコード例

```markdown
<!-- _class: content -->
<!-- _header: "セクション名" -->

## スライド見出し

- 箇条書き項目1
- 箇条書き項目2
- 箇条書き項目3
```

### 容量制限

| 要素 | 上限 |
|:-----|:-----|
| h2 | {{CONTENT_H2_LIMIT}}文字 |
| 箇条書き | 最大{{CONTENT_LIST_LIMIT}}項目 |
| 1項目 | {{CONTENT_ITEM_LIMIT}}文字（1行） |

---

## `summary` — まとめスライド

**用途**: プレゼンのまとめ（番号付きリスト）
**背景**: グラデーション
**テキスト色**: 白

### Marpコード例

```markdown
<!-- _class: summary -->

# まとめ

## サブタイトル

1. ポイント1
2. ポイント2
3. ポイント3
```

### 容量制限

| 要素 | 上限 |
|:-----|:-----|
| h1 | {{SUMMARY_H1_LIMIT}}文字 |
| リスト | 最大{{SUMMARY_LIST_LIMIT}}項目 |
| 1項目 | {{SUMMARY_ITEM_LIMIT}}文字（1行） |

## 画像レイアウトクラス

既存パターン内で `<div>` に適用するレイアウトクラス:

| クラス | 配置 |
|:-------|:-----|
| `img-text-right` | テキスト左 + 画像右 |
| `img-text-left` | 画像左 + テキスト右 |
| `img-center` | 画像中央 + キャプション |
| `img-full` | 全幅ダイアグラム |

### 使用例

```markdown
<!-- _class: content -->

## 説明テキスト

<div class="img-text-right">
<div>

- テキスト内容
- 箇条書き

</div>
<img src="images/diagram.svg">
</div>
```
