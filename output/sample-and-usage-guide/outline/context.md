# コンテキスト定義

## 発表/公開の場
個人の技術ブログ（Zenn / はてなブログ / note など）

## 聴衆/読者
- Claude Code を使い始めたが、スキル機能をまだ活用していないエンジニア
- LTや技術記事を書きたいが「何から始めればいいかわからない」と感じている人
- AIとの対話で文章を書くワークフローに興味がある人

## 制約条件
- 文字数: 3,000〜5,000字程度
- フォーマット: Markdown（見出し・コードブロック含む）
- 読了時間: 5〜10分

## その場で求められること
- 実際に使えるツールの紹介なので、具体的な使い方が伝わること
- 「自分も試してみよう」と思える導線があること
